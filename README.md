# Walmart-Near-Real-Time-Data-Warehouse-using-HYBRIDJOIN
Developed a near-real-time Walmart Data Warehouse using Star Schema, HYBRIDJOIN, and optimized ETL pipelines in Python to process and enrich large-scale transactional retail data. Implemented OLAP analytics, fact/dimension modeling, stream processing, and batch loading in MySQL for scalable business intelligence and  sales analysis
